## Vectara

Vectara is not strictly a vector database, rather it's an end-to-end GenAI platform.
In order to use Vectara as a vector database with EmbedChain, 
first [sign-up](https://vectara.com/integrations/embedchain) for a free account if you don't have one already, 
and then set the environment variables 
`VECTARA_CUSTOMER_ID`, `VECTARA_OAUTH_CLIENT_ID` and `VECTARA_OAUTH_SECRET` 
(see [here](https://docs.vectara.com/docs/learn/authentication/oauth-2) for how to obtain those).

<CodeGroup>
```python main.py
from embedchain import App

# load Vectara configuration from yaml file
app = App.from_config(config_path="config.yaml")
```

```yaml config.yaml
app:
  config:
    id: my_app
vectordb:
  provider: vectara
  config:
    collection_name: vectara-demo
    allow_reset: true
embedder:
  provider: fake
chunker:
  chunk_size: 1000000
  chunk_overlap: 0
```
</CodeGroup>
