`reset()` method allows you to wipe the data from your RAG application and start from scratch.

## Usage

```python
from embedchain import Pipeline as App

app = App()
app.add("https://www.forbes.com/profile/elon-musk")

# Reset the app
app.reset()
```

<Note>
If you would like to only delete the chat history and keep the data sources, you can use the `delete_chat_history` method.
```python
app.delete_chat_history()
```
</Note>
