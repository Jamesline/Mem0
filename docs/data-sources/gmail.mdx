To use GMAILLoader you must install the extra dependencies with `pip install --upgrade embedchain[dataloaders]`.

To load GMAIL messages, use the data_type as `gmail`. Since it is hard to automatically detect, it is advised to specify the `data_type` when adding a GMAIL document.
The next argument must be a valid email address. Eg:

```python
import os
from embedchain.apps.app import App

os.environ["GOOGLE_CRED_PATH"] = 'credentials.json'

app = App()

emails = ['deven@embedchain.ai', 'deshraj@embedchain.ai']
app.add(", ".join(emails))
app.query("Summarize my email conversations")
```
